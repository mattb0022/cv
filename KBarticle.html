Yealink Troubleshooting

Yealink W60B base LEDs – The base station has 3 LEDs that can be used to help troubleshoot issues

  1. Registration - After at least one handset registered to the base station, the registration LED illuminates solid green. Follow instructions below for pairing a handset.
  2. Power – This LED should be solid green if the base station has power. If LED is off, please ensure that it is connected to a PoE switch or power adapter is connected.
  3. Network – If this LED is flashing, the base station is not connected to a network or cannot get an IP address. It will turn solid green once it has connected to a network. Try connecting the base station to another switch port, if the network indicator still slowly flashes, try factory resetting the base.

Reset to factory settings

This step is not required for brand new out-of-the-box phones; however, if the phone has been already used then it is necessary.

  • Press OK button on Your handset.
  • Go to Settings menu.
  • Go to System Settings submenu and select Base Reset option.
  • Enter Your PIN. NOTE: Default PIN is 0000.
  • Select Reset to factory option.
  • Confirm with OK button on prompt.
  • Wait sometime prior for device to reboot.

After you attempt these solutions, if the Yealink is still experiencing issues, escalate ticket to GSD Support

Registering Handset to the Yealink Base

To register a new handset manually:

When the handset LCD screen prompts “Press base page 2s then press Reg.”, long press on the base station till the registration LED flashes.

 

Easy Registration:

Press the Reg soft key on the handset to register quickly.

 

Normal Registration:

Press the OK soft key on the handset, and then select Register Handset.

Select the desired base and then press the OK soft key. The handset begins searching the base.

Press the OK soft key after searching a base successfully.

Enter the base PIN 2287, and then press the Done soft key to complete registration.

After the success of registration, the handset LCD screen prompts “Handset Subscribed” and “Base NO. (The last 4 characters of connected Base’s MAC address)”. After initializing data successfully, an icon with internal handset number and handset name appears on the LCD screen.

 

Note: If the handset LCD screen prompts “Searching for Base”, please check if your base station is powered on.

 

Cisco MPP 8811 Troubleshooting

 

 

 

 

Verify Network Connectivity 

 

  1. Press Applications 
  1.  Select Status > Network Status

You can view the following information: (NOTE ALL OF THIS)

  • Network type—Indicates the type of Local Area Network (LAN) connection that the phone uses.
  • Network status—Indicates if the phone is connected to a network.
  • IPv4 status—IP address of the phone. You can see information on IP address, Addressing type, IP status, Subnet mask, Default router, Domain Name Server (DNS) 1, DNS 2 of the phone.
  • VLAN ID—VLAN ID of the phone.
  • MAC address—Unique Media Access Control (MAC) address of the phone.

 If the phone is not getting an IP address or the network status show disconnected, follow the steps to verify the network port is functional.

Verifying The Network Port is Functional
  1. Exchange the Ethernet cables with cables that you know are functional.
  2. Disconnect a functioning Cisco IP Phone from another port and connect it to this network port to verify that the port is active.
  3. Connect the Cisco IP Phone that does not start up to a different network port that is known to be good.
  4. Connect the Cisco IP Phone that does not start up directly to the port on the switch, eliminating the patch panel connection in the office.

 

Verify the Phone Has Power
  1. If you are using external power, verify that the electrical outlet is functional.
  2. If you are using in-line power, use the external power supply instead.
  3. If you are using the external power supply, switch with a unit that you know to be functional.

If the phone still does not start up properly, power up the phone with the handset off-hook. When the phone is powered up in this way, it attempts to launch a backup software image. If the phone still does not start up properly, perform a factory reset of the phone.

Factory Resetting Cisco MPP 8811 Phone
  1. Remove power from the phone in one of these ways:
  • Unplug the power adapter.
  • Unplug the LAN cable.
  1. Wait 5 seconds.
  2. Press and hold # and plug the phone back in.
  3. When the Headset and Speaker buttons are lit, release # and enter the following key sequence:

123456789*0#

The light for the Headset button turns off after you press the 1 key. After you enter the key sequence, the Mute button lights.

Caution: Do not power down the phone until it completes the factory reset process, and the main screen appears.
